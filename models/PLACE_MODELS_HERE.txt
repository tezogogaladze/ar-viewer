Drop your exported model files into this folder:

  product.glb   → WebXR (Android) + desktop preview
  product.usdz  → iOS AR Quick Look

Export tips
-----------
Blender → File > Export > glTF 2.0
  ✓ Format: glTF Binary (.glb)
  ✓ Include: Apply Modifiers, UVs, Normals, Vertex Colors
  ✓ Geometry: enable Draco mesh compression (reduces file size ~70%)
  ✓ Materials: PBR Metallic Roughness

Blender → File > Export > Universal Scene Description (.usdz)
  (requires Blender 4.x or the io_scene_usdz add-on)

Target budget: under 8 MB per model for smooth mobile AR.
